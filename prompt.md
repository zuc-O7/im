# Role
你是一位資深的 Front-end Developer 兼 UI/UX 設計師，擅長使用現代化、極簡且具有質感的 Web 技術。

# Task
請為我建立一個單頁式的「個人作品集網站 (Portfolio)」。請確保程式碼結構清晰，且 HTML, CSS, JavaScript 邏輯分離。

# Technical Stack
- HTML5 (語意化標籤)
- CSS3 (使用 Flexbox/Grid 佈局，禁止使用外部 Framework 如 Tailwind，以便我學習原生 CSS)
- Vanilla JavaScript (原生腳本)

# Design Style
- **風格關鍵字**：[例如：極簡主義、深色模式 (Dark Mode)、Bauhaus 風格、Apple 般的質感]
- **配色方案**：[例如：背景 #0f172a, 主色調 #38bdf8, 文字 #f8fafc]
- **字體**：使用無襯線字體 (sans-serif)，標題要有粗體張力。
- **互動**：加入細微的 Hover 動效，以及捲動時的淡入效果 (Reveal on scroll)。

# Sections to Include
1. **Hero Section**：包含我的名字 [你的名字]、一句震撼的標語 (Slogan) 以及一個聯絡按鈕。
2. **About**：簡短介紹我的背景與專業。
3. **Projects**：以卡片式佈局展示三個作品，每張卡片要有圖片佔位符、標題、描述與技術標籤。
4. **Skills**：展示我擅長的技術（如：HTML, CSS, JS, Python）。
5. **Footer**：社群連結與版權宣告。

# Constraint
- 請確保網站是響應式設計 (Responsive Design)，在手機上也能完美呈現。
- JavaScript 需包含一個簡單的互動功能（例如：點擊按鈕切換深淺色模式，或導覽列的平滑捲動）。
- 所有的 CSS 需寫在獨立的 style.css，JS 寫在 script.js。
